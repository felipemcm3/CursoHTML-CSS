Explicação

Você pode representar as cores no css por nome, hexadecimal, rgb que é binario e hsl.
Todas as cores são combinações das 3 primarias vermelho,verde e azul.
Os tons de cada cor e definido pela numeração do hexadecimal ou binario ou hsl.O visual studio code também permite uma seleção de cores manual.
