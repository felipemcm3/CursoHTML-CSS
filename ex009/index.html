Criando listas no HTML

Use a tag ol para criar listas ordenadas e li para os itens da lista.,
Você pode alterar o tipo de marcação usando o parametro type em ol.

  1. 1 para numeros
  2. A letras maiusculas
  3. a letras minusculas
  4. I números romanos maiusculos
  5. i números romanos minusculos
